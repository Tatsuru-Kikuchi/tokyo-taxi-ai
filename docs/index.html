📚 使い方ガイド

お客様向け

アプリを起動
全国AIタクシーアプリを開き、「お客様」モードを選択します
位置情報を許可
正確な配車のため、位置情報の使用を許可してください
目的地を設定
地図上で目的地をタップ、または住所を入力します
料金を確認
AI予測による料金見積もりを確認し、配車をリクエスト
配車完了
ドライバーの到着をお待ちください。LINE通知も利用可能です

ドライバー向け

ドライバーモードを選択
アプリ起動後、「ドライバー」モードを選択します
オンラインに切り替え
営業開始時は「オンライン」ボタンをタップ
AI推奨エリアを確認
需要予測マップで高需要エリアを確認できます
リクエストを受諾
配車リクエストが来たら、詳細を確認して受諾
収益を最大化
AI収益予測を活用して、効率的な営業を実現

よくある質問

位置情報はどのように使用されますか？

位置情報は配車サービスの提供にのみ使用され、お客様のプライバシーは厳重に保護されます。

料金はどのように計算されますか？

基本料金に加え、距離、時間、天候、需要などをAIが総合的に判断して算出します。

LINE連携は必須ですか？

LINE連携は任意です。連携すると配車状況の通知を受け取れて便利です。
