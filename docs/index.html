❓ よくある質問

位置情報はどのように使用されますか？
位置情報は配車サービスの提供にのみ使用されます。お客様の位置情報は暗号化されて送信され、配車完了後は自動的に削除されます。第三者への提供は一切行いません。
料金はどのように計算されますか？
基本料金に加え、距離、時間、天候、需要などをAIが総合的に判断して算出します。料金は配車前に見積もりとして表示され、透明性を確保しています。
LINE連携は必須ですか？
LINE連携は任意です。連携すると配車状況の通知を受け取れて便利ですが、連携しなくてもアプリ内で全ての機能をご利用いただけます。
キャンセル料はかかりますか？
配車確定後5分以内のキャンセルは無料です。それ以降のキャンセルには、ドライバーの移動コストを考慮したキャンセル料が発生する場合があります。
